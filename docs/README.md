# Fantacom Programming Guide
So, you want to try out this new fantasy computer whose design could change at any commit? You've come to the right place!

Since the boot ROM is still not usable as a BIOS yet, this guide will focus on bare-metal programming. You'll be initializing the memory map, the VGA, and everything by yourself. Fun stuff!

Also, I'll be assuming that you know Z80 assembly and assemblers. Either that, or how to hack together a CRT0 file like I did with Z88DK ZCC.
